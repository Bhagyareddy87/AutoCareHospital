# AutoCareHospital
# 🏥 Healthcare Admission Data Analysis using Microsoft SQL Server

## 📌 Project Overview

This project analyzes hospital admission and discharge data using Microsoft SQL Server. The dataset was cleaned, duplicates were removed, and healthcare metrics were calculated to generate meaningful insights.

## 🛠️ Technologies Used

* Microsoft SQL Server
* SQL Server Management Studio (SSMS)
* T-SQL

## 🧹 Data Cleaning

✔ Removed duplicate records using ROW_NUMBER()
✔ Used CTE for data cleansing
✔ Filtered NULL MRD numbers
✔ Created a reusable view (vw_AdmissionData)

## 📊 Analysis Performed

* Total Discharges
* Average Daily Discharge Rate
* Average Length of Stay (ALOS)
* Discharge Distribution by Age
* Discharge Distribution by Age Group
* Discharge Distribution by Gender
* Discharge Distribution by Day of Week

## 💡 SQL Concepts Used

* Views
* Common Table Expressions (CTE)
* ROW_NUMBER()
* PARTITION BY
* CASE Statements
* Aggregate Functions
* GROUP BY
* ORDER BY
* CAST
* ROUND
* DATEPART
* FORMAT

## 🎯 Key Insights

* Cleaned and standardized healthcare data.
* Identified discharge trends.
* Analyzed patient demographics.
* Generated healthcare performance metrics.
